> PixMe is a web application for generating a QRCode to send a pix to a random user.
> Currently it only sends to me...

### Motivation and Concepts
The motivation for developing this project started when i randomly asked some of my colleagues to send me a pix of a random value, if they wanted. For my surprise some of them did send me a considerably ammount, so the idea of making an application to receive a pix from random people started.

The concept of the application is based on the user generating a QRCode that serves as a link to the pix payment method used in Brazil, when scanning the QRC the pix screen appears and is up to the user confirm it.
As a method for encouraging people to confirm the transaction, there is a leaderboard of transactions displaying below the QRC area.

I have a lot of new features in mind which will be coming soon, including the possibility to create an account and register your pix key, so when someone generates a QRC your's might be the lucky one.

### Future plans

The project is still in development and the next updates will be focusing on the following tasks:
